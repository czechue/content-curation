# Content Curation

Multi-agent content curation system built with Gas Town.

## Overview

Tracks 20+ content sources (YouTube, podcasts, blogs) using parallel AI agents,
rates content with Fabric patterns, and generates weekly curated digests.

## Stack

- **Gas Town** - Multi-agent orchestration
- **Beads** - Task/memory tracking
- **Fabric** - Content rating patterns
- **Obsidian** - Output destination
